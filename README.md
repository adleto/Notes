Simple and easy to use app for creating and editing Notes.

Features:
* simple interface
* creating & editing notes
* ecrypted storage (locally and on cloud)
* sync notes between different devices
* encrypted backups on cloud

Runs on:
* [Android]()
* [UWP (Windows 10)]()
* [iOS]()
